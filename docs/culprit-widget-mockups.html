Held constant across every candidate

  • Fail-safe honesty (B-156 G1 generalized): no surface ever assumes an event into existence. An unconfirmed slot is a visible gap, not a ✓.
  • Provenance: every widget write carries logged_via='widget' (B-289 lands first — unbackfillable later).
  • Never reassures: no “all clear,” no “Biscuit is doing great,” no Signal/AI-read copy in v1 (a widget's refresh cadence can leave a clinical sentence stale on the Home Screen — deliberately deferred, see session notes).
  • No monetization state (D5 parity with Home): the widget never badges, upsells, or caps.
  • Silent by design: a widget is the anti-nag surface — zero notifications, Principle 4 untouched.
  • One accent: teal on interactive elements only; everything else neutral (or the night palette if ratified).
