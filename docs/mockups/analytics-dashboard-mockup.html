The §11 safety invariants, made visible

InvariantWhere you see it in v2
#1 · Intake is not preferenceCard ⑦ titled “Meals finished,” meals-only; copy “treats aren't counted here…it would hide the real signal.” A decline routes to a vet watch, never “picky.”
#2 · n=1 never reassuresNo card wears a reassuring colour off one sample. Falling vomits state count + direction and stop. The baseline read (2b) compares, never declares wellness.
#3 · Adverse counts honest, never gamifiedCard ② — falling vomits = calm slate, not green; muted sparkline, neutral number. No streaks/badges.
#4 · Associational only, non-causal cueNo card asserts a cause. The Signal hedges (“since the switch,” “early pattern”). Meals&treats notes coverage, not blame.
#5 · Minimum-sample floorsEvery rate shows its denominator (39 of 50; 24 of…). Below floor → the “still learning Nyx's baseline” state, never a fabricated rank.
#6 · Free-feeding honestyTop food is “showed up in N meals,” not “ate most.” Meals&treats is what was logged, never read as what was eaten.
