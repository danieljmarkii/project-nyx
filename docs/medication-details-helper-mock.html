B-160 — Medication details: helper copy + popular-name chips

The manual “Medication details” screen (app/medication-capture.tsx, manual / AI-confirm path). Mock shows the active pet as cat “Nyx”, so the name chips lead with feline-relevant drugs. Two states below: (A) name empty → the new chips + helper copy appear; (B) a chip has filled the name → chips vanish, strength shows its format helper and the unchanged confirm gate.

The one clinical catch (spec §3): example chips go on the drug name only. Strength gets format helper copy, never value chips — a tappable “5 mg / 10 mg” would erode the §6.5 dose-confirm gate (picking a chip feels like confirming it). Strength is the one field the owner must read off their own label.
